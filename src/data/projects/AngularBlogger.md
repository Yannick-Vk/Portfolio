A simple front-end project for my blogging platform created with **Angular**.
Where users can register and log in to create and read blog posts.
The blog content is written in **Markdown** format and rendered using the `ngx-markdown` library.
The project is styled using **Tailwind CSS** for a modern and responsive design.

## Features

- User Authentication: Secure login and registration system.
- Blog Management: Users can create and read blog posts.
- Markdown Support: Blog content is rendered from Markdown for easy editing.
- Role Management: Administrators can view and manage user roles.
- Responsive Design: Styled with Tailwind CSS for a modern, responsive UI.

## Tech Stack

- Framework: Angular (latest, v20+)
- Language: TypeScript 5.9
- Styling: Tailwind CSS 4
- State Management: Angular Signals & RxJS
- Markdown: ngx-markdown for rendering
